# Resume App

This is a Resume app made using Vue.js framework

## Requirements
 1. Docker
 2. Docker-compose
 3. API service please run `https://github.com/shinjiescorido/nest-resume`
## Installation
```bash
git clone https://github.com/shinjiescorido/vue-resume.git
cd vue-resume
docker-compose up -d
```
## Stack
1. Vue.js
2. Webpack
3. Eslint
4. Docker
5. Sass
